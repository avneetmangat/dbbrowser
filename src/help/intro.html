DBBrowser


DBBrowser is an open source (GPL license), cross-platform tool which can be used to view the contents of a database. It supports CLOBS and BLOBS. It is designed to work with Oracle and MySQL. The user should never have to write SQL to view the data although a SQL window is provided. Support for ER (Entity Relationship) diagrams, Oracle XMLType, other databases(IBM DB2, PostGreSQL) is planned for the next version.

Main features:

  1. View all data and sequences in tables without writing SQL
  2. Add filters to a table to view a subset of data in the table
  3. Support for CLOBs and BLOBS in Oracle and MySQL
  4. SQL syntax highlighting and SQL code help
  5. Add, remove, update records without writing SQL
  6. Add, remove columns from tables
  7. Export data as HTML, XML, CSV or PDF docs
  8. Reverse engineer DDL from existing DB schema
  9. More than 10 GUI skins to change the look and feel
  10. Context Sensitive Help
  11. Internationalized with 2 language packs. Support for plugging in more language packs

Other features:

  1. Cross platform
  2. Cross platform installer, Windows exe installer and rpm for Linux for ease of installation
  3. Documentation with javadocs, Help files, Technical architecture document, UI specs and Class diagrams

Planned for next version:

  1. Support for Entity Relationship diagrams
  2. Support for Oracle XMLTypes
  3. Support for PostGreSQL and IBM DB2

DBBrowser is hosted on the SourceForge website (http://sourceforge.net/projects/databasebrowser).
