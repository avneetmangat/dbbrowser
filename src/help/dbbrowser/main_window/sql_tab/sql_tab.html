SQL tab

This panel allows the user to run SQL statements. An SQL syntax colourer is built in to colour SQL keywords. The colour mapping is:
SQL keyword Colour
Select Blue
As Blue
Distinct Blue
Insert Red
Update Red
Alter Red
Drop Red
Grant Red
Set Red
( Magenta
) Magenta
Values Dark Gray
Where Dark Gray
Into Dark Gray
From Dark Gray
. Red
, Red

The algorithm to colourize the SQL keywords is not efficient and will be improved in the next version. It is auto-disabled for very large SQL statements.


DBBrowser also provides Code help. Press CTRL+space to bring up code help. It might take some time to index the contents of the database while the code help is being built.

Once the indexing has been done, press CTRL+space to bring up a list of schemas, tables and column names. These are shown in an heirarchical order allowing the user to choose any value to insert into the SQL statement.
