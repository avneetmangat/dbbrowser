Connection Information window

Connection information is used to manage connections to databases. This window allows one to create a new connection to a database, change the connection details and delete a connection. To connect to a database, the user has to specify:
  • Name of the connection - to identify this connection
  • Location of type 4 JDBC driver file - DBBrowser connects to the database using this vendor specific JDBC driver. Download this from the vendors website
  • Name of Driver class - this class implements the java.sql.Driver interface. DBBrowser recursively scans the JDBC jar file and lists all the drivers in the jar file
  • URL for use by the JDBC driver
  • Username
  • Password


Set up a connection in the New Connections panel and click on 'Connect'.
